# UC Berkeley AI Hackathon

## Setup

1. run the backend
    - Mac users use `source venv/bin/activate`
    ```
    cd backend
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt
    pip freeze > requirements.txt
    ```

2. run the frontend
    ```
    cd frontend
    npm i
    npm run dev
    ```
